【整体要求】
-------------------------
1. 输出内容必须是纯 Markdown，不得包含代码块符号或非 Markdown 格式。

4. 若正文中包含图片，请在原位置插入 Markdown 图片标签。
5. 所有链接必须使用完整 URL 格式（含 https://）。
6. 文末信息的顺序必须固定为：发推时间 → 推文数据 → 原推链接。

若正文中任意位置有图片，插入：

引用推文整体必须使用一个完整的区块引用（>），并在正文结束后一行空行后呈现。

>  
> {quoted_text}  
>  
> （若引用推文包含图片，则在原位置插入，不要遗漏引用推文中的图片）

>  
> - **引用推文链接：** [{quoted_url}]({quoted_url})
引用推文内部的所有 Markdown 行必须带 >，缩进必须完全一致。
